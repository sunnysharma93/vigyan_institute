# Notes Folder

This folder contains study notes files for Vigyan Institute students.

## File Structure

Place your PDF, DOC, and DOCX files in this folder. The files should match the filenames specified in the `src/data/notes.ts` file.

## Supported File Types

- PDF (.pdf)
- Microsoft Word (.doc, .docx)

## Example Files

The following files are referenced in the notes data:

- physics-mechanics-fundamentals.pdf
- chemistry-organic-compounds.pdf
- mathematics-calculus-basics.pdf
- physics-electromagnetic-induction.pdf
- chemistry-chemical-bonding.pdf
- mathematics-trigonometry.pdf
- physics-thermodynamics.pdf
- chemistry-periodic-table.pdf
- mathematics-probability.pdf
- physics-wave-optics.pdf
- chemistry-acids-bases.pdf
- mathematics-coordinate-geometry.pdf

## Important Notes

1. Ensure all files are properly named and match the data in `notes.ts`
2. Files should be optimized for web (compressed)
3. Use descriptive filenames with hyphens instead of spaces
4. Keep file sizes reasonable for faster downloads

## Deployment

This folder will be deployed as-is to the public directory, making files accessible via:
`/notes/filename.pdf`
